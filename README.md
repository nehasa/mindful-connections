# Mental Health & Stress Coaching Website

## Overview

A comprehensive mental health and stress coaching website built with HTML, CSS, and JavaScript. The website features a modern, responsive design with light pastel colors and focuses on helping individuals achieve true happiness through meaningful connections.

## Website Structure

### Pages Created

1. **Homepage** (`index.html`)
   - Hero section with motivational headline and subtext
   - Free trial class section
   - Features section highlighting key services
   - Navigation with smooth scrolling

2. **Programs Page** (`programs.html`)
   - Four comprehensive programs priced at Rs.19,900 each
   - Program details, duration, and features
   - Online delivery format
   - Program selection functionality

3. **My Coaching Philosophy Page** (`philosophy.html`)
   - Focus on spirituality and ancient wisdom
   - Integration of modern psychology with spiritual practices
   - Personal journey and approach explanation

4. **Who I Help Page** (`who-i-help.html`)
   - Happiness framework based on three pillars: Luck, Action, Emotional Strength
   - Image placeholder section for user customization
   - Target audience identification

5. **Contact Page** (`contact.html`)
   - Comprehensive contact form
   - Multiple contact methods (phone, email, WhatsApp)
   - FAQ section
   - Quick contact options
